# capabilities

## Purpose

Tauri capability definitions that control which app features and permissions are granted to the frontend.

## Key Files

| File | Description |
|------|-------------|
| `default.json` | Default capability granting opener plugin access to the frontend |

## For AI Agents

### Working In This Directory

- Capabilities use JSON schema defined by Tauri 2
- Each capability specifies which permissions the webview has
- Modifying capabilities requires restarting the Tauri dev server
