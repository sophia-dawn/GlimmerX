# assets

## Purpose

Static image and SVG assets imported by React components via Vite's asset handling.

## Key Files

| File | Description |
|------|-------------|
| `react.svg` | React logo used in the default App.tsx scaffold |

## For AI Agents

### Working In This Directory

- Import via `import logo from '@/assets/react.svg'` — Vite returns the processed URL
- SVGs can also be imported as React components via `?component` suffix if needed
- Add new assets here only if they are actual files (not icons, which go in `public/`)
