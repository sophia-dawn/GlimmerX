# 01 - 项目概述

> 返回 [DESIGN.md](../DESIGN.md)

## 项目定位

**GlimmerX** 是一款基于 Tauri + React 构建的跨平台个人复式记账桌面应用。
核心理念：**本地优先、数据私有、复式记账、简洁高效**。

## 设计哲学

借鉴三款成熟项目的优势：

| 借鉴对象 | 借鉴内容 |
|---------|---------|
| **Beancount** | 数据模型：严格复式记账（每笔交易借贷平衡）、纯文本账本设计理念 |
| **Firefly III** | 功能架构：账户体系（资产/负债/收入/支出）、预算管理、报表、规则自动化 |
| **Actual Budget** | 交互体验：本地优先、快速响应、信封预算、现代简洁 UI |

## 技术栈

| 层级 | 选型 | 说明 |
|---|---|---|
| 桌面壳 | **Tauri 2** (Rust) | 轻量、安全、跨平台，二进制体积小 |
| 前端框架 | **React 19** + TypeScript | 组件化 UI |
| 构建工具 | **Vite** | 快速 HMR |
| UI 组件 | **shadcn/ui** + Tailwind CSS | 源码级组件，高度定制 |
| 数据库 | **SQLCipher** (SQLite 加密扩展) | 数据库文件 AES-256 加密，用户密码解锁 |
| 表格 | **TanStack Table** | 灵活的头组件，支持排序/过滤/分页 |
| 图表 | **Recharts** | 收支统计、趋势可视化 |
| 金额格式化 | **react-number-format** | 千分位、小数位、货币符号 |
| 日期处理 | **date-fns** | 轻量、Tree-shakeable |

## 项目边界

### 做什么（In Scope）

- ✅ 个人/家庭级别的复式记账
- ✅ 桌面端应用（Windows / macOS / Linux 三平台一等公民）
- ✅ 三平台功能、UI、交互完全一致
- ✅ 纯本地数据存储（SQLCipher 加密 SQLite）
- ✅ 数据库密码解锁（类似 KeePass）
- ✅ 多账户管理、交易录入、分类、预算、报表
- ✅ 数据导入/导出（CSV、Beancount 格式）
- ✅ 定期交易、暗色/亮色主题、数据备份与恢复

### 不做什么（Out of Scope）

- ❌ 多用户协作、云端同步、银行 API 对接
- ❌ 多币种汇率实时换算、企业财务功能
- ❌ 移动端、账本分享/社交功能
