## 核心交互

### 账本解锁/创建流程

```
应用启动
   ├─ 检查最近列表中是否有已知数据库
   ├─ 有 ─→ [创建新账本] / [打开已有账本] / [最近列表]
   └─ 无 ─→ 仅显示 [创建新账本]
```

- **新建账本**: 文件保存对话框 → 输入密码（二次确认）→ SQLCipher `PRAGMA key` 加密
- **打开账本**: 文件打开对话框 → 输入密码 → SQLCipher 解密验证
- **锁定**: 清除内存中的数据库连接，返回解锁页

### 快速记账弹窗

- 全局快捷键 `Ctrl/Cmd + N` 触发
- 简化录入：金额 + 分类 + 备注
- 自动推导分录：`Expenses:XXX` 借方 + `Assets:默认账户` 贷方

### 交易列表

- 按日期分组的卡片式时间线
- 悬停显示操作按钮，顶部搜索栏支持全文搜索

### 预算页面

- 月度视图，进度条，超支红色高亮，底部总结余

#### 交互规则

- **收支卡片**：收入显示绿色 `+`，支出显示红色 `-`，数值由后端计算
- **财务健康卡片**：资产/债务显示绝对值，净资产为差值（负数红色警示）
- **月度收支图**：按天聚合，当天有收入显示绿色柱，有支出显示红色柱；无交易日期不显示
- **分类收支聚合**：饼图或柱状图展示本月各分类收支占比；点击分类跳转分类详情
- **支出 Top 10**：当月金额最大的 10 笔支出交易（交易级别，不按分类聚合），按金额降序；点击跳转交易详情页
- **最近交易**：最多显示 15 条，按日期倒序；空状态显示"暂无交易记录"
- **账户余额列表**：默认展开，可点击折叠；按账户类型分组显示
- **跳转行为**：点击"查看全部"跳转 `/transactions`；点击交易行跳转详情页；点击账户名跳转 `/accounts/:id`

## 国际化（i18n）

### 技术方案

| 项目 | 选型 |
|---|---|
| 核心库 | **i18next** + **react-i18next** |
| 状态管理 | **Zustand** |
| 翻译存储 | `src/i18n/locales/{lng}.json` |
| 语言持久化 | `localStorage` 键名 `glimmerx:language` |
| 自动检测 | `navigator.language` → 回退 `en` |

### 支持语言

| 代码 | 名称 | 优先级 |
|---|---|---|
| `en` | English | P0 (默认/回退) |
| `zh` | 简体中文 | P0 |

### 翻译命名空间

`common`、`unlock`、`onboarding`、`dashboard`、`accounts`（含 wizard/types/typeDescriptions 子命名空间）、`navigation`、`placeholder`

使用方式：`t("accounts.title")`，支持插值 `{{name}}` 和富文本 `<1>文本</1>`。
